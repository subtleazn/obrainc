# Contributing to the OBRA Inc. Blogger Theme

We welcome contributions from the community! Thank you for your interest in improving this theme and helping non-profit organizations.

## How to Contribute

There are many ways to contribute to this project. You can:

*   **Report Bugs:** If you find a bug, please open an issue on our GitHub repository. Please include a detailed description of the bug, including steps to reproduce it.
*   **Suggest Features:** If you have an idea for a new feature, please open an issue on our GitHub repository. We would love to hear your ideas!
*   **Submit Pull Requests:** If you would like to contribute code to the project, please submit a pull request.

## Pull Request Guidelines

*   Please make sure that your code adheres to the existing code style.
*   Please include a detailed description of your changes in the pull request.
*   Please make sure that your changes are well-tested.

Thank you for your contributions!
